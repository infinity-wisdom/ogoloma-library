# Ogoloma Library Management System

A full-featured library management web application designed with a Corporate/Modern Academic aesthetic.

## Pages

### Public Pages
- `index.html` — Homepage with hero section and category browser
- `search.html` — Book search and results
- `book-details.html` — Individual book detail view
- `login.html` — Member login
- `register.html` — New member registration

### Member Pages
- `dashboard.html` — Member personal dashboard

### Admin Pages
- `admin.html` — Admin dashboard with analytics
- `manage-books.html` — Book catalog management
- `add-book.html` — Add new books
- `manage-users.html` — User/member management
- `circulation.html` — Borrowing and returns management
- `reports.html` — Reports and analytics

## Design System
- **Colors:** Deep Navy (`#022448`) primary, Gold/Amber (`#795900`) accent
- **Typography:** Inter (all weights)
- **Framework:** Tailwind CSS (CDN)
- **Icons:** Google Material Symbols
